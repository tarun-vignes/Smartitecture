## Current Scope

- Windows 10 and Windows 11 beta only.
- General AI answers route through the hosted Smartitecture backend.
- Local tools cover performance, running processes, battery, network, Defender scan/status, and app launching.
- Sensitive or destructive automation should require explicit user confirmation.
- Basic local logs are written to `%LOCALAPPDATA%\Smartitecture\Logs`.

## Tester Report Checklist

Ask testers to include:

- Windows version.
- Smartitecture version.
- Steps to reproduce.
- Expected result.
- Actual result.
- Screenshot or recording for UI issues.
- `%LOCALAPPDATA%\Smartitecture\Logs\app.log`.
- `%LOCALAPPDATA%\Smartitecture\Logs\audit.log` when the issue involves diagnostics or app launching.

## Known Limits

- Mac and Linux are not supported in this beta.
- Public translation quality still needs review before claiming broad language support.
- Render free tier can cold-start after inactivity.
- The MSIX is development-signed for beta testing.
- Windows may keep an old pinned taskbar icon until Smartitecture is unpinned and repinned.
- Release assets are served from the website download directory for the current beta.
