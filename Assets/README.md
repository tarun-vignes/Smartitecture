# Smartitecture Assets

## Logo Files

### Main Logo
- `smartitecture-logo.png` - Official Smartitecture logo 
  - Beautiful hexagonal network design with security and chip elements
  - Blue to purple gradient color scheme
  - High-resolution PNG format for crisp display

### Usage Guidelines
- Use the PNG format for crisp display at any size
- Maintain the gradient colors for brand consistency
- Extract design elements for mode indicators:
  - Security elements for FORTIS mode
  - Performance elements for NEXA mode
  - Intelligence elements for LUMEN mode

### Integration Points
- Main application header
- Mode selection indicators
- Loading screens and splash screens
- About dialogs and branding elements
